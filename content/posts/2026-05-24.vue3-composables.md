## 什么是 Composable

Composable 是 Vue 3 Composition API 中的核心概念——将有状态逻辑提取到可复用的函数中。

```typescript
// 一个简单的计数器 Composable
export function useCounter(initialValue = 0) {
  const count = ref(initialValue)
  const increment = () => count.value++
  const decrement = () => count.value--

  return { count, increment, decrement }
}
```

## 设计原则

### 1. 单一职责

每个 Composable 应该只负责一件事。如果它变得过于复杂，应该拆分成更小的 Composable。

### 2. 返回 Ref 和 Computed

```typescript
// ✅ 好的做法
export function useMouse() {
  const x = ref(0)
  const y = ref(0)
  // ...
  return { x, y }
}

// ❌ 不好的做法 — 返回裸值
export function useMouse() {
  let x = 0
  // x 变化时不会触发响应式更新
  return { x }
}
```

### 3. 清理副作用

```typescript
export function useEventListener(target, event, handler) {
  onMounted(() => target.addEventListener(event, handler))
  onUnmounted(() => target.removeEventListener(event, handler))
}
```

### 4. 使用 `useAsyncData` 获取数据

在 Nuxt 中，使用 `useAsyncData` 或 `useFetch` 来利用 SSR 和缓存优势。

## 总结

编写好的 Composable 不仅能让代码更简洁，还能提高可测试性和可维护性。
