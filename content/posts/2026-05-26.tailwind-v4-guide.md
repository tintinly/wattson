## Tailwind CSS v4 新特性

### CSS-first 配置

v4 最大的变化是采用 CSS-first 配置方式：

```css
@import "tailwindcss";

@theme {
  --color-primary: #3b82f6;
  --font-sans: 'Inter', sans-serif;
}
```

不再需要 `tailwind.config.js`！

### 零配置暗色模式

```css
/* 自动支持 dark: 变体 */
.dark .my-element {
  background: var(--color-gray-900);
}
```

### 性能提升

- 使用 Rust 引擎 (Lightning CSS)，构建速度提升 10 倍
- JIT 模式下零配置
- 更小的 CSS 体积

## 迁移步骤

1. 升级依赖
2. 移除 `tailwind.config.js`
3. 使用 `@import "tailwindcss"` 替代
4. CSS 变量替代 `theme.extend`
5. 更新 `content` 路径配置

## 注意事项

v4 与 v3 的大部分 utility class 兼容，但插件系统有较大变化，需要确认第三方插件是否支持。
